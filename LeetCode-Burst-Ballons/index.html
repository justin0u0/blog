LeetCode 312 - Burst Ballons

Posted on 2021-05-23 In LeetCode Views: Disqus:

題目

題目連結：https://leetcode.com/problems/burst-balloons/

給一長度為 n 的序列 nums，依照任意順序刪除一個數字，直到所有數字消失。

每次刪除一個數字 x 時，假設 x 的左邊為 y，右邊為 z，則花費 x*y*z 元；當左邊沒有數字，y 視為 1，當右邊沒有數字時，z 視為 1。

求出最大花費。

想法

首先，若想要利用 DP 來紀錄重複的狀態，期望上的狀態應該為 dp(i,j) 代表將 nums[i:j] 依照任意順序刪除直到所有數字消失的最大花費，並且希望 dp(i,j) 的答案可以由一個在 i∼ji\sim ji∼j 的子區間的 DP 值求得。

可以試圖枚舉在 i∼ji\sim ji∼j 之間要刪除的數字，但是會導致狀態無法表述當前剩下的數字。例如，初始狀態的 dp(0,n-1) 若刪除數字 1 則剩下的數字為 [3,5,8]，然而如此一來就沒有一個狀態能夠表述了。因此想辦法改變轉移的方式。觀察範例測資：

為了避免刪除數字時，可能沒有左邊或右邊的數字的問題，先在整個序列的兩側加上兩個 1，並把題目改為將序列內所有除了兩側的數字刪除的最大花費。

再度觀察，可以發現枚舉一個區間內要刪除的數字會使得狀態無法表述，那麼反過來改為枚舉一個區間內最後一個被刪除的數字，就有點像將上述的順序倒過來。可以發現轉移式即為：

dp(i,j)=max(dp(i,k),dp(k,j))+nums[i]×nums[k]×nums[j]∀ k∈[i+1,j−1]dp(i,j)=max\big(dp(i,k), dp(k,j)\big)+nums[i]\times nums[k]\times nums[j]\quad\forall\ k\in[i+1,j-1] dp(i,j)=max(dp(i,k),dp(k,j))+nums[i]×nums[k]×nums[j]∀ k∈[i+1,j−1]

也就是，要求出 i∼ji\sim ji∼j 的最大花費，且不包含 i, ji,\ ji, j，若最後一個被刪除的數字為 kkk，則一定會有 nums[i]×nums[k]×nums[j]nums[i]\times nums[k] \times nums[j]nums[i]×nums[k]×nums[j] 之花費，並且再加上 i∼ki\sim ki∼k 與 k∼jk\sim jk∼j 之花費，分別是 dp(i,k)dp(i,k)dp(i,k) 與 dp(k,j)dp(k,j)dp(k,j)。

如此一來，狀態有 O(N2)O(N^2)O(N2) 中，每一個狀態需要花 O(N)O(N)O(N) 的時間轉移，因此得到一個總時間複雜度為 O(N3)O(N^3)O(N3) 的 DP。
