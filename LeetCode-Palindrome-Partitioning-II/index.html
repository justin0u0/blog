LeetCode 132 - Palindrome Partitioning II

發表於 2021-05-23 分類於 LeetCode Disqus：

題目

給定一個字串 s，找出最少需要將字串 s 切成幾段使得每一段都是迴文。

想法

要將字串 s 切分，使得每一段都是迴文。可以發現要求 s[0:n-1] 的最小迴文切割段數，可以枚舉任何一段是迴文 s[j:n-1] 的，將 s[j:n-1] 當做是一段迴文，再接著求 s[0:j-1] 之最小迴文切割段數即可。

有了上述的想法，可以發現這是一個 DP 問題，因為答案可以由某一個子答案中求得。

因此，首先定義狀態 dp(i) 代表將 s[0:i] 切段使得每一段都是迴文所需的最小段數。 則轉移為：

dp(i)=min⁡(dp(j)+1)∀ j<i, if s[j+1:i] is palindromedp(i)=\min\big(dp(j)+1\big)\quad\forall\ j\lt i,\ if\ s[j+1:i]\text{ is palindrome} dp(i)=min(dp(j)+1)∀ j<i, if s[j+1:i] is palindrome

因此剩下的問題變成，如何很快的求得任何一段 s[j+1:i] 是否為迴文。利用預處理的方式，可以很簡單的在 O(N2)O(N^2)O(N2) 的時間求出每一段子字串是否為迴文，並儲存進一個二維陣列 isPalindrome[i][j] 代表 s[i:j] 是迴文字串。主要有兩個做法：

  1. 枚舉每一個位置做為迴文的中心點（注意偶數長度迴文的情況），往外擴展（左右各往外一格字元，如果相等的話）直到不能在擴展。如此一來只會有 O(N)O(N)O(N) 個中心點且每個中心點最多往外擴展 O(N)O(N)O(N) 次，因此總時間複雜度為 O(N2)O(N^2)O(N2)。
  2. 利用 DP 的方式，可以發現：

isPalindrome(i,j)=1if s[i]=s[j]&isPalindrome(i+1,j−1)=1isPalindrome(i,j)=1\quad if\ s[i]=s[j]\And isPalindrome(i+1,j-1)=1 isPalindrome(i,j)=1if s[i]=s[j]&isPalindrome(i+1,j−1)=1

因此只要從小區間到大區間依次算出答案即可。
